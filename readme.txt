thinkphp 版本 3.2.3

简单说明
	微信功能篇：
		基本设置：绑定公众号
		关键字回复：设置关键字回复各基础回复类型都可以
		自定义菜单：同官网，可以设置“点击”关键字回复
		微信菜单：这个开发用的，一般不要动
		素材管理：上传删除的都是永久素材，不提供临时素材接口，同时不能编辑图文素材消息（以后单独增加）,删除时会删掉文件
		用户列表：新绑定的可以获取微信端所有用户数据，再次关注或者其它操作时两端可同步，有备注、换组功能
		用户组：同上，具有创建、修改、删除分组功能
		清理缓存：清除上一个绑定的公众号的token等数据，不换公众号用不到
		清理用户：清理所有用户，暂时用不到

目录：
app
  |-common  	公共配置文件及公用函数
  |-Index   	前台模块
  	|-Session  	前台session存放目录
  |-Mzsj    	后台管理模块（后台使用mysql存放session）
  |-Uc			Ucenter文件
  |-Runtime 	thinkphp生成的缓存文件
Core			ThinkPHP框架文件（修改了U方法，可以路由后的参数直接解析出来）
Static			静态文件存放地址
Upload			上传文件存放的地址
index.php  		前台入口绑定文件
mzsj.php   		后台入口绑定文件
.htaccess		伪静态配置文件
